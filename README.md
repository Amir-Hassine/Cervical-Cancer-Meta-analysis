# Cervical cancer meta-analysis
This repository contains the R code used to perform the statistical analyses
for the systematic review and meta-analysis on cervical cancer risk.
## Contents
- `MA_analysis_git.R`: main analysis and figures for both forest plots and funnel plots when applicable 
## Data
The code assumes summary-level data as described in the manuscript.
File paths in the R scripts are provided as relative paths.
## Software
Analyses were conducted using R (version 4.3.3).
